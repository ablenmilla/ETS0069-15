 1 len()
Returns the length of the string.

text = "hello"
print(len(text))  # Output: 5

2 str.encode()
Encodes a string into bytes.

text = "hello"
print(text.encode())  # Output: b'hello'

3 str.islower()
Returns True if all characters are lowercase.

text = "hello"
print(text.islower())  # Output: True
